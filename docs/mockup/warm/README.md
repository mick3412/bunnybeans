# 暖色／在地感風格示範

米色底、暖棕主色、深綠成功、磚紅警示。適合餐飲、選物店或希望介面有「在地、溫暖」感的品牌。

## 視覺重點

- 主背景：米白 #F7F3ED；卡片與頂欄 #FFFDF9、暖灰邊 #d4c9bc
- 側欄：深棕 #3d3229、奶油色字；選中左線 #c4a574（暖金）
- 主按鈕／連結：暖棕 #6b5344
- 成功：深綠 #4a7c59；警示：磚色 #a0522d

## 開啟

```bash
open docs/mockup/warm/index.html
open docs/mockup/warm/pos.html
```
