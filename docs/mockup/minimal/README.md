# 極簡／留白風格示範

大量白與極淺灰、單一主色（黑）、裝飾極少、字階清楚、間距大。適合強調「專業、不干擾」的後台或報表為主情境。

## 視覺重點

- 主背景：#fafafa；側欄與卡片白底、細灰邊 #e5e5e5
- 主按鈕／選中：黑 #1a1a1a（單一主色）
- 連結：黑字＋底線
- 成功／警示：綠 #15803d、紅 #b91c1c（語意保留、不搶眼）

## 開啟

```bash
open docs/mockup/minimal/index.html
open docs/mockup/minimal/pos.html
```
