# 儀表板／數據感風格示範

深色側欄＋淺色主區；KPI 以少量鮮色（藍／綠／橙／灰）區分；表格與卡片邊框極細。適合以報表、儀表板為主的後台，強調數據可讀性。

## 視覺重點

- **側欄**：深灰藍 #1e293b、選中左線 #38bdf8
- **主區**：淺灰 #f1f5f9；頂欄與卡片白底
- **KPI 卡**：左側色條區分 — 第 1 張藍 #0ea5e9、第 2 張綠 #16a34a、第 3 張橙 #ea580c、第 4 張灰 #64748b
- **表格**：邊框 1px #e2e8f0、隔行 #fafbfc、hover #f1f5f9
- **主按鈕／連結**：#0ea5e9
