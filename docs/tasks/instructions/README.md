# INSTRUCTIONS（僅保留最新）

本資料夾存放「每輪任務指令」的**最新版本**檔案。為避免檔名尾碼編號造成連結失效，本 repo 其它文件一律連到本資料夾而非單一檔名。

## 使用規則（協作流程用）

- **最新版本**：本資料夾只保留一組檔案（後端/前端各一份）。若仍採用編號，則以 **資料夾內唯一存在**的 `BACKEND-INSTRUCTIONS NNN.md` 與 `FRONTEND-INSTRUCTIONS NNN.md` 為準（例如目前為 `003`）。
- **最新版本**：本資料夾只保留一組檔案（後端/前端各一份）。若仍採用編號，則以 **資料夾內唯一存在**的 `BACKEND-INSTRUCTIONS NNN.md` 與 `FRONTEND-INSTRUCTIONS NNN.md` 為準（以資料夾內最新檔為準）。
- **規格 Agent（執行 agent-collab）更新方式**：
  1. 讀取本資料夾中目前唯一一組（或最新編號 `NNN`）的兩份檔案。
  2. 依協作流程只更新兩份檔案的 **§1**（其餘段落不動）。
  3. 更新完成後，**產生新檔**：將編號改為 `NNN + 1`（例如 `003 → 004`），並將更新後內容寫入新編號檔案。
  4. **刪除舊檔**：確保資料夾內永遠只保留最新一組檔案。
- **Backend / Frontend Agent**：在各自對話窗中，請 @ `docs/tasks/instructions/` 後，開啟最新編號檔案，只讀 **§1** 進行實作。
