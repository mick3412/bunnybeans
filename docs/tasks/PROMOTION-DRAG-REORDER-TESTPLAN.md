## 促銷規則拖曳排序（reorder/bulk）手測清單

### 前置條件
- 後端可連線（`VITE_API_BASE_URL` 正確）
- 促銷規則列表至少有 3 筆

### 步驟
1. 進入 `/admin/promotions`
2. 以左側「拖曳把手」拖動某一筆到不同位置後放開
3. 觀察：
   - UI 立即更新順序（optimistic）
   - 右側顯示 `排序中…`
   - 成功後 toast 顯示「已更新排序」
4. 重新整理頁面確認排序仍保留（後端已落盤）
5. 故障/權限/後端回錯時：
   - toast 顯示錯誤訊息
   - UI 會回滾到原本排序
